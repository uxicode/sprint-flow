# 📅 Workflow - Jira 업무 보고서 자동화 대시보드

이 프로젝트는 Jira 티켓 데이터를 분석하고 간편하게 일일/주간 업무 보고서용 마크다운 문서를 생성해 주는 웹 대시보드 프로토타입입니다.

## 🚀 주요 기능

- **다크 모드 & Glassmorphism 디자인**: 시각적으로 세련되고 몰입감 있는 개발자 친화적 디자인 시스템 적용.
- **동적 Jira JQL 빌더**: 프로젝트 키, 시작/종료일, 담당자 입력을 바탕으로 Jira 표준 쿼리(JQL)를 자동으로 빌드합니다.
- **고품질 Mock 데이터 시뮬레이터**: 실제 Jira 계정이 없어도 가상의 고품질 개발 티켓 데이터를 생성하여 서비스를 즉시 테스트할 수 있습니다.
- **업무 보고서 자동 생성 템플릿**:
  - **일일 업무 보고서**: 팀원별로 오늘 완료한 일, 진행 중인 일, 내일 예정된 일을 정리합니다.
  - **주간 업무 보고서**: 완료율 요약 표, 팀원별 상세 작업 목록, 다음 주 계획을 생성합니다.
- **원시 티켓 데이터 조회**: 가공되지 않은 티켓 상세 리스트를 테이블 뷰로 확인 가능합니다.
- **간편한 내보내기**: 원클릭으로 마크다운 텍스트를 클립보드에 복사하거나 `.md` 파일로 다운로드할 수 있습니다.

## 💻 실행 방법

본 프로젝트는 순수 HTML, CSS, JavaScript로 개발되어 별도의 빌드 과정 없이 즉시 실행 가능합니다.

### 방법 1. 브라우저에서 바로 열기
1. 프로젝트 폴더의 `index.html` 파일을 더블클릭하여 크롬, 사파리 등의 브라우저에서 엽니다.

### 방법 2. 로컬 웹 서버 실행 (권장)
로컬 웹 서버 환경에서 열면 브라우저 보안 제약 없이 더 안정적으로 동작합니다.
터미널을 열고 프로젝트 폴더 내에서 다음 중 하나의 명령을 실행하세요.

* **Python 사용 시:**
  ```bash
  python3 -m http.server 8000
  ```
  이후 브라우저에서 `http://localhost:8000`으로 접속합니다.

* **Node.js 사용 시:**
  ```bash
  npx -y serve
  ```
  이후 화면에 표시되는 로컬 URL(예: `http://localhost:3000`)로 접속합니다.

## 🛠️ Jira API 연동 및 CORS 우회 가이드

### 1. Jira API Token 발급 방법
1. [Atlassian API Tokens](https://id.atlassian.com/manage-profile/security/api-tokens) 페이지에 접속하여 로그인합니다.
2. **Create API token** 버튼을 클릭하여 이름을 지정하고 토큰을 생성합니다.
3. 생성된 토큰 값을 안전한 곳에 복사해 둡니다. (이 툴의 사이드바 'Jira API 설정'에 입력합니다)

### 2. 브라우저 CORS 문제 해결 방법
웹 브라우저에서 Jira Cloud API를 직접 호출할 경우, Jira 서버의 CORS(Cross-Origin Resource Sharing) 보안 정책에 의해 호출이 차단될 수 있습니다. 프로토타입에서 실제 데이터를 연동하려면 아래 방법 중 하나를 사용해야 합니다.

#### 방법 A: 크롬 CORS 해제 확장 프로그램 사용 (가장 간단함)
- Chrome 웹스토어에서 **"Allow CORS: Access-Control-Allow-Origin"** 또는 유사한 CORS 허용 확장 프로그램을 설치하고 활성화합니다.

#### 방법 B: CORS 프록시 서버 사용
- 로컬 또는 클라우드에 간단한 proxy(예: `cors-anywhere`)를 띄운 뒤, 요청 URL 앞에 프록시 주소를 붙여 요청을 우회합니다.
